# FeedForward

FeedForward is an LLM-powered pipeline that analyzes Intercom support conversations to extract product themes and create implementation-ready stories.

## Run the web app

1) Install dependencies
```bash
npm install
```

2) Start the API server
```bash
python -m src.api.main
```

3) Start the web app
```bash
cd webapp
npm install
npm run dev
```

Then open `http://localhost:3000`.
