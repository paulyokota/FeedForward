# Tailwind Product Taxonomy

Product context for theme extraction and classification.

## Core Product Areas

### 1. Scheduling & Publishing

- **Pin Scheduler** - Schedule pins to Pinterest
- **Multi-network scheduling** - Publish to Pinterest, Instagram, Facebook simultaneously
- **Browser Extension** - Schedule from anywhere on the web
- **Video scheduling** - Support for .mov/.mp4, 3-60 seconds, 100MB max
- **Cross-posting** - Instagram to Pinterest sync

### 2. Content Creation

- **Tailwind Create** - Design tool for visual content
- **Ghostwriter AI** - AI copywriting for captions/descriptions
- **Canva Integration** - Direct publishing from Canva

### 3. Platform Integrations

- **Pinterest** (primary) - Pins, boards, analytics
- **Instagram** - Posts, reels, stories
- **Facebook** - Pages, posts
- **E-commerce** - Etsy, eBay, Shopify, TeachersPayTeachers

### 4. Communities

- **Tailwind Communities** - Collaborative pinning groups
- **Content sharing** - Community-based content discovery

### 5. Analytics & Research

- **Pinterest SEO** - Keyword research
- **Performance tracking** - Pin/post analytics

### 6. Account Management

- **Multi-profile dashboard** - Manage multiple accounts
- **Team/agency features** - Collaboration tools
- **Billing** - Plans, subscriptions, payments

## Common Issue Patterns

Based on support structure, these are likely issue clusters:

| Area             | Common Issues                                     |
| ---------------- | ------------------------------------------------- |
| **Auth/Connect** | Pinterest/Instagram disconnection, OAuth failures |
| **Scheduling**   | Posts not publishing, wrong time, timezone issues |
| **Content**      | Image/video format errors, size limits            |
| **Billing**      | Plan changes, refunds, feature access             |
| **Communities**  | Join/leave issues, content not appearing          |
| **Extension**    | Browser compatibility, not loading                |

## Feature → User Intent Mapping

| When user mentions... | They likely mean...                          |
| --------------------- | -------------------------------------------- |
| "pins not posting"    | Scheduling → Pinterest publishing failure    |
| "can't connect"       | Auth → OAuth/API connection issue            |
| "wrong board"         | Scheduling → Board selection/routing         |
| "community"           | Communities → Tailwind Communities feature   |
| "create" / "design"   | Tailwind Create → Content creation tool      |
| "extension"           | Browser Extension → Chrome/Firefox add-on    |
| "smart schedule"      | SmartSchedule → Auto-optimized posting times |
